# TidyTier Product Website

Professional GitHub Pages website for the TidyTier kitchen organizer brand.

## Upload to GitHub
Upload all files to the root of your `tidytier` repository.

## GitHub Pages
Settings → Pages → Deploy from branch → main → /(root)

## Live URL
https://tidytier.github.io/tidytier/

## Important
Replace `support@tidytier.com` with your real support email if different.
